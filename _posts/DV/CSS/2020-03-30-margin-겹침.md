css를 먹일 때 요소들이 붙어있는 경우 margin을 주었지만 먹히지 않을 때 해결방법을 알아본다.

이 문제는 margin이 겹칠 경우 작은 값을 가지는 요소의 margin을 무시하는 것이다.

요소에 `display: inline-block;`을 걸어주면 된다.

```css
.class {
  display: inline-block;
}
```
