# 장점

* 시각적인 UI 구성 확인
* 뷰컨트롤러의 흐름 확인
* 지저분한 단순 UI attiribute 코드를 스토리보드로 이관해서 코드양을 줄일 수 있음


# 단점

* 직접 생성자 호출해서 초기화하는 것이 아니기 때문에 생성자를 통한 초기값 설정 불가능
* 파일이 커지면 속도 저하 극심
* 대규모 개발시 컨플릭이 매우 빈번하게 발생
* 20~30명의 프로젝트를 하게되면 현실적으로 사용불가
