# 생각

쿼리로 가져온 table에 대해서 추출을 진행할 때, 사용할 수 있는 테크닉이다. 이건 예제로 보는 것이 정확하다.

# Code

```sql
SELECT ANIMAL_ID, NAME,
        CASE WHEN
            (SEX_UPON_INTAKE LIKE "Neutered%" OR SEX_UPON_INTAKE LIKE "Spayed%") THEN "O"
            ELSE "X"
        END AS "중성화"
    FROM ANIMAL_INS
    ORDER BY ANIMAL_ID;
```
